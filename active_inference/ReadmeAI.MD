## Public API surface (umbrella)

Umbrellas do not expose a single API ‚Äî each app exports its own. See the per-app `ReadmeAI.MD` tables for the full public surface. High-traffic entry points:

- [`AgentPlane.Runtime`](apps/agent_plane/lib/agent_plane/runtime.ex) ‚Äî start/step/perceive/plan/act a Jido agent
- [`WorldPlane.Engine`](apps/world_plane/lib/world_plane/engine.ex) ‚Äî create/advance an episode (GenServer)
- [`WorkbenchWeb.Episode`](apps/workbench_web/lib/workbench_web/episode.ex) ‚Äî orchestrates agent + world across the Markov blanket
- [`WorldModels.Bus`](apps/world_models/lib/world_models/bus.ex) ‚Äî subscribe to event streams
- [`WorldModels.EventLog`](apps/world_models/lib/world_models/event_log.ex) ‚Äî append / query the Mnesia event log
- [`ActiveInferenceCore.DiscreteTime`](apps/active_inference_core/lib/active_inference_core/discrete_time.ex) ‚Äî pure POMDP math
- [`ActiveInferenceCore.Equations`](apps/active_inference_core/lib/active_inference_core/equations.ex) ‚Äî equation registry

## Telemetry / signals / events emitted here

Full taxonomy in [`apps/agent_plane/lib/agent_plane/telemetry/bus.ex:17-32`](apps/agent_plane/lib/agent_plane/telemetry/bus.ex). Summary:

- `[:active_inference_core, :discrete_time, :call, :start|:stop|:exception]` ‚Äî pure math span
- `[:jido, :agent_server, :signal, :start|:stop|:exception]` ‚Äî Jido signal lifecycle
- `[:jido, :agent_server, :directive, :start|:stop|:exception]` ‚Äî Jido directive lifecycle
- `[:jido, :agent, :cmd, :start|:stop|:exception]` ‚Äî Jido cmd lifecycle

Re-emitted as `WorldModels.Event` of type `"equation.evaluated"` or `"runtime.jido.*"` by [`AgentPlane.Telemetry.Bus`](apps/agent_plane/lib/agent_plane/telemetry/bus.ex).
